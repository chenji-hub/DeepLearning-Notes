# 01_深度学习基础理论笔记

> 本笔记主要梳理深度学习的底层逻辑与二分分类的前置数学知识。

## 1. 神经网络的核心概念
* **基础单元（神经元）**：以房价预测为例，输入数据经过一个单一神经元（可以视为一个函数，如 **ReLU** 激活函数），输出预测结果。
* **神经网络**：将许多单一神经元互相连接、按层次堆叠。隐藏层能够自动从输入数据中提取复杂的特征。
* **驱动力**：深度学习爆发的三大要素是**海量数据 (Data)**、**强大算力 (Computation)** 和 **算法创新 (Algorithms)**。

## 2. 逻辑回归 (Logistic Regression) 与二分分类
* **任务目标**：判断输入数据属于哪一类（例如：预测图片是否为猫，y=0 或 y=1）。
* **数据表示**：图片等高维数据会被展平成一维的特征向量 $x$。
* **线性计算**：计算输入特征与权重的线性组合，加上偏差：
  $$z = w^T x + b$$
* **非线性激活**：为了将结果转化为 $0$ 到 $1$ 之间的概率，使用 Sigmoid 函数进行映射：
  $$\sigma(z) = \frac{1}{1 + e^{-z}}$$
* **预测公式**：
  $$\hat{y} = \sigma(w^T x + b)$$

> **核心参数说明**：
> * **$w$ (Weights/权重)**：衡量每个输入特征的重要性。
> * **$b$ (Bias/偏差)**：用于对结果进行基础平移。
> 在模型未经训练时，它们是随机数；训练的过程就是寻找最优 $w$ 和 $b$ 的过程。
